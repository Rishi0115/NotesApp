# 📒 AI Notes App

* Full-stack Notes App using React, Node.js, and MongoDB
* Create, update, delete, and manage notes
* Upload PDFs and extract content
* AI-powered Q&A using Gemini (strict context-based answers)
* Secure JWT authentication system
* Email OTP verification for users
* Profile management (update info, profile picture)
* Modern UI with glassmorphism and responsive design
* Clean, scalable, production-ready architecture
